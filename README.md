# qpu-assembler2

An assembler for VideoCore IV QPU.


## Compiling

```
$ make
```


## Usage

Use https://github.com/Terminus-IMRC/qpu-bin-to-hex to convert the output to C/C++'s array style.

```
$ qasm2 <in.qasm2 | qbin2hex
```
